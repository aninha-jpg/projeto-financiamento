Este projeto foi desenvolvido como atividade prática da disciplina de Programação Orientada a Objetos(POO), do curso Análise e Desenvolvimentos de Sistemas da PUCPR.
Com o objetivo principal de ser um sistema de financiamento de imóveis, como: casas, terrenos e apartamentos.

O sistema encapsula alguns fundamentos como classess e objetos, classes abstratas, herança, polimorfismo, tratamento de exceções e serialização.

IDE usada: Eclipse.

Status: Projeto concluído e utilizado somente para fins acadêmicos.
Data: 17/11/2024.
